# RCADS-47 Youth English Questionnaire Recreation

This project is a React-based web application that attempts to recreate the RCADS-47 Youth English questionnaire, originally published by UCLA (see https://www.childfirst.ucla.edu/wp-content/uploads/sites/163/2018/03/RCADS47-Youth-English-2018.pdf).

## Features

- Displays the questionnaire statements in batches of 20 questions.
- Allows users to select frequency responses: Never, Sometimes, Often, Always.
- Supports adding additional batches of questions until all are answered.
- Provides an option to export all responses to the clipboard.

## Usage

1. Install dependencies:
   ```
   npm install
   ```

2. Start the development server:
   ```
   npm start
   ```

3. Open the app in your browser at `http://localhost:3000`.

## Notes

This is a simple recreation intended for demonstration and educational purposes only.
